## Repository Structure

### English
The repository is organized into independent layers:

- Repository Layer
- Standards Layer
- Foundation Layer
- Theory Layer
- Research Layer
- RFC Layer
- Specification Layer
- Reference Layer

Each layer has a distinct responsibility and is connected through canonical references.

仓库由多个独立层组成：

- Repository Layer（项目层）
